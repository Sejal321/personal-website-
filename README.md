# Sejal Sahni Personal Website

A modern, resume-driven personal website focused on impact metrics, engineering experience, and project highlights.

## Project structure

- `index.html` - page content and sections
- `main.css` - styles and responsive layout
- `main.js` - mobile nav + footer year behavior
- `scripts/serve.mjs` - tiny local static server
- `scripts/smoke-test.mjs` - simple validation script

## Quick start

```bash
npm test
npm start
```

Then open `http://localhost:5173`.

## Customize content

Update these sections in `index.html`:

- Hero and intro copy
- Experience cards
- Featured project details
- Skills and education
- Contact links
